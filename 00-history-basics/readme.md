# History of Linux

## The Origins (1991)

In 1991, **Linus Torvalds**, a 21-year-old student at the University of Helsinki, became frustrated with the licensing and limitations of **MINIX** (a small Unix-like system used for teaching). On August 25, 1991, he posted a now-famous message to a Usenet newsgroup:

- UNIX is a powerful, professional operating system created in the 1960s that set the architectural blueprint for modern computing but was often expensive and proprietary. MINIX was a smaller, simplified version developed later by a professor for teaching purposes, serving as the educational foundation that inspired Linus Torvalds to create Linux.
- UNIX is a full Operating System, not just a kernel.

  > _"I'm doing a (free) operating system (just a hobby, won't be big and professional like gnu) for 386(486) AT clones..."_

Initially, he wanted to call it **Freax** (a mashup of "free," "freak," and "x" for Unix), but the administrator of the server where he hosted the code renamed the folder to **Linux** without his permission. The name stuck.

## Key Milestones

### 1. The GPL and the GNU Project (1992)

Linux was just a **kernel**—the core "brain" that manages hardware. To be a full operating system, it needed tools (compilers, shells, editors). **Richard Stallman** and the **GNU Project** had already built these tools but lacked a working kernel. In 1992, Torvalds released Linux under the **GNU General Public License (GPL)**. This ensured the code would remain free for anyone to use, modify, and distribute.

### 2. The Rise of Distributions (1993–1994)

Because installing Linux was incredibly difficult, "distributions" (distros) emerged to bundle the kernel with software and installers:

- **Slackware (1993):** The oldest surviving distribution.
- **Debian (1993):** Known for its commitment to free software and the basis for Ubuntu.
- **Red Hat (1994):** Brought Linux into the enterprise world.

### 3. The Corporate Shift (Late 90s - 2000s)

In 1998, major tech companies like **IBM, Oracle, and HP** announced support for Linux. This moved Linux from a "hacker hobby" to a serious competitor for Microsoft Windows in the server room.

### 4. The Mobile and Cloud Era (2008–Present)

- **Android (2008):** Google released Android, which is built on the Linux kernel. This made Linux the most widely used OS in the world by sheer volume.
- **The Cloud:** Today, Linux powers over **90%** of the public cloud workload (AWS, Azure, GCP) and **100%** of the world’s top 500 supercomputers.

---

## Why Linux Succeeded

- **Open Source:** Thousands of developers worldwide contribute to it, making it incredibly secure and stable.
- **Portability:** It runs on everything from tiny smart light bulbs to massive space station computers.
- **Cost:** Being free to use allowed startups and tech giants to scale without massive licensing fees.
