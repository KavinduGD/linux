# Linux Course Notes

This repository contains my personal notes and learnings from a Linux course. It covers various topics ranging from basic file management to advanced shell scripting, user management, and Vim usage.

## What I Have Learned

Through this course, I have gained a solid understanding of:

- **Linux History & Fundamentals**: The origins of Linux and differences between distributions.
- **Text Editing**: Proficient usage of Vim, including modes, navigation, editing, and visual block commands.
- **File Management**: Essential commands for creating, removing, copying, and moving files and directories (`mkdir`, `rm`, `cp`, `mv`).
- **Data Streams**: Understanding stdin, stdout, stderr, and how to redirect them.
- **Environment Variables**: Managing and using environment variables effectively.
- **Shell Scripting**: Basics of shell scripting, custom prompts, and shell expansions.
- **File System**: Deep dive into Linux file systems, inodes, hard links, and soft links.
- **User Management**: Permissions, access control, and user administration.
- **SSH**: Secure remote access and key management.
- **Systemd**: Service management and system initialization.
- **Processes**: Understanding zombie processes and process lifecycle.
